# Blob Storage
A high-performance, responsive Cloud File Management Dashboard built with Nuxt 4, Tailwind CSS, and Pinia. Designed with a sleek, premium dark-mode aesthetic, delivering intuitive experience for modern file organization and uploading

## Quick Start
```
# Clone the project
git clone https://github.com/kinnaamf/blob-storage-frontend.git

# Install dependencies
npm install

# Start development server
npm run dev
```
